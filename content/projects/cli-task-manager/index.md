A modern command-line interface for task management that combines simplicity with powerful features. Built with Rust for performance and reliability, this tool integrates seamlessly into developer workflows.

## Features

- Fast, intuitive command interface
- Local JSON-based storage with optional encryption
- Advanced filtering and search capabilities
- Priority levels and due date tracking
- Calendar integration (Google Calendar, iCal)
- Cross-platform support with consistent experience

This project showcases my ability to build user-friendly CLI tools with careful attention to developer experience. The codebase demonstrates Rust best practices including error handling, memory safety, and performance optimization.
